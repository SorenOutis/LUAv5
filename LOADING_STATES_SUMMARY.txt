================================================================================
                         HOW IT WORKS
================================================================================

LOADING STATE DETECTION:
  1. Each page uses `usePage()` from Inertia.js
  2. Computed property `isLoading` checks page.props.loading
  3. While page data loads, skeleton appears
  4. Once data loads, skeleton is replaced with actual content

IMPLEMENTATION PATTERN:

  // In script
  import { usePage } from '@inertiajs/vue3';
  import SkeletonStats from '@/components/SkeletonStats.vue';
  import SkeletonCard from '@/components/SkeletonCard.vue';
  
  const page = usePage();
  const isLoading = computed(() => page.props.loading === true);

  // In template
  <SkeletonStats v-if="isLoading" :count="4" />
  <div v-else class="grid gap-4 md:grid-cols-4">
    <!-- Actual content -->
  </div>

   - Props: class (string)

2. SkeletonCard.vue (Card Layout)
   - Simulates card with title, description, buttons
   - Multiple placeholders per card
   - Props: count (number, default: 1)
   
   Usage: <SkeletonCard :count="6" />

3. SkeletonStats.vue (Stats Grid)
   - 4-column responsive grid
   - Each stat has label and value skeleton
   - Props: count (number, default: 4)
   
   Usage: <SkeletonStats :count="4" />

================================================================================
                         STYLING & ANIMATION
================================================================================

SKELETON STYLING:
  - Base color: Tailwind's bg-slate-200 (light) / bg-slate-800 (dark)
  - Animation: Tailwind's animate-pulse class
  - Rounded corners: rounded-md
  - Responsive: Adapts to dark mode automatically

ANIMATION:
  - Smooth pulsing effect (opacity: 0.7 to 1.0)
  - Duration: ~2 seconds
  - Continuous loop until content loads
  - CPU-friendly (uses CSS animations, not JavaScript)

DARK MODE:
  - Skeleton.vue uses dark:bg-slate-800 for dark mode
  - SkeletonCard/Stats inherit dark mode from Skeleton
  - All skeleton elements are automatically dark mode compatible
