ROUTING LAYER (Inertia.js)
==========================

Protected Routes:
├── GET /courses → CoursesController@index
├── GET /quests → QuestsController@index
├── GET /leaderboard → LeaderboardController@index
├── GET /achievements → AchievementsController@index
├── GET /progress → ProgressController@index
├── GET /rewards → RewardsController@index
├── GET /messages → MessagesController@index
└── GET /profile → UserController@profile

Middleware:
├── 'auth' - User must be authenticated
└── 'verified' - User email must be verified

├── UserController.php
│   └── profile() - Display user profile

│   └── index() - List courses with filters

Each Controller:

DATA FLOW
=========

    ↓

    ↓
Laravel Controller
    ├─ Get user
    ├─ Fetch data (sample or DB)
    └─ Format for Vue
    ↓

    ├─ Receive props
    ├─ Process data
    └─ Render HTML
    ↓
Browser Displays Page


PAGE FEATURES
=============

COURSES PAGE:
├── Filter: Enrolled | Completed | Available
├── Cards: Course name, instructor, progress, XP
├── Difficulty: Beginner | Intermediate | Advanced
├── Progress: Visual bars with percentages
└── Layout: Responsive grid (1-3 columns)

QUESTS PAGE:
├── Filter: Active | Completed | Available
├── Cards: Quest title, objective, difficulty, reward
├── Progress: Tracked for active quests
├── Countdown: Days remaining for active quests
└── Icons: Emoji-based difficulty indicators

LEADERBOARD PAGE:
├── Filter: By XP | Level | Streak | Achievements
├── Podium: Top 3 with special styling
├── Rankings: Full list with ranks 4+
├── Highlight: Current user marked
└── Sorting: Dynamic by selected metric

ACHIEVEMENTS PAGE:
├── Filter: All | Unlocked | Locked
├── Display: Grid with emoji icons
├── Rarity: Percentage shown
├── Status: Grayscale for locked, color for unlocked
└── Modal: Click for full details

PROGRESS PAGE:
├── Filter: By category (Learning, Assessments, etc.)
├── Level: Current level with XP to next level
├── Metrics: Multiple progress bars by category
├── Milestones: Timeline of recent achievements
└── Stats: Completion rates and learning time

REWARDS PAGE:
├── Badges: Grid display (5 columns)
├── Rarity: Color coding (Common/Uncommon/Rare/Legendary)
├── Rewards: List with type badges
├── Milestones: Progression toward next reward
└── Share: Buttons to share achievements

MESSAGES PAGE:
├── Filter: All | Unread | Notifications | Announcements
├── Types: System, Notification, Message, Announcement
├── Status: Unread messages highlighted
├── Preview: First line of message content
└── Modal: Click to view full message

PROFILE PAGE:
├── Header: User info with avatar
├── Stats: XP, Level, Courses, Assignments, Streak
├── Progress: Course progress visualization
├── Achievements: Recent unlocked badges
├── Edit: Dialog to update profile
